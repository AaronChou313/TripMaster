## 基本操作

### 导航
应用包含四个主要功能模块，可通过顶部导航栏切换：
- **地点管理**：搜索和管理旅行景点
- **行程规划**：创建和优化旅行路线
- **预算管理**：跟踪和分析旅行开支
- **备忘事项**：记录旅行相关信息和提醒

### 数据保存
所有数据都会自动保存到本地，无需手动保存。关闭浏览器后重新打开，数据依然存在。

## 地点管理

### 搜索景点
1. 在搜索框中输入景点名称或关键词
2. 点击搜索按钮或按回车键
3. 结果会在左侧列表和地图上同时显示

### 管理兴趣点
- **添加兴趣点**：点击搜索结果旁的"+"按钮
- **移除兴趣点**：在右侧列表中点击"×"按钮
- **查看详情**：点击兴趣点可在地图上高亮显示

## 行程规划

### 创建行程
1. 点击"新建行程"按钮
2. 输入行程名称和描述
3. 从兴趣点列表中拖拽景点到行程中
4. 调整景点顺序以优化路线

### 编辑行程
- **添加景点**：从兴趣点列表拖拽到行程中
- **调整顺序**：在行程列表中拖拽景点重新排序
- **编辑详情**：点击景点可添加描述、预算等信息
- **删除景点**：点击景点旁的删除按钮

### 路线预览
- 切换到预览模式可查看完整的行程路线
- 地图上会显示景点间的连接路线
- 支持打印行程路线图

## 预算管理

### 预算同步
1. 点击"从行程同步"按钮
2. 系统自动计算行程中所有景点的预算总和
3. 按类别汇总显示各项支出

### 手动添加预算
- **交通费用**：往返机票、当地交通等
- **住宿费用**：酒店、民宿等住宿成本
- **餐饮费用**：每日餐费预算
- **门票费用**：景点门票和其他活动费用
- **购物费用**：纪念品、特产等购物预算
- **其他费用**：保险、通讯等其他支出

### 预算分析
- **饼图**：显示各类预算占比
- **柱状图**：对比不同类别的支出金额
- **总计显示**：实时显示预算总额

## 备忘事项

### 创建备忘录
1. 点击左侧"新建备忘录"按钮
2. 系统会创建一个名为"新备忘录"的空白文档
3. 点击备忘录可开始编辑

### 编辑功能
#### 基本文本编辑
- 在标题输入框中输入备忘录标题
- 在内容区域输入详细内容
- 支持中文、英文等多种语言

#### Markdown语法支持
备忘录支持标准Markdown语法，包括：

**文本格式**
```
**粗体文本**
*斜体文本*
`行内代码`
```

**标题层级**
```
# 一级标题
## 二级标题
### 三级标题
```

**列表**
```
- 无序列表项1
- 无序列表项2

1. 有序列表项1
2. 有序列表项2
```

**链接和图片**
```
[链接文字](网址)
![图片描述](图片地址)
```

**代码块**
```
​```javascript
console.log('Hello World');
​```
```

#### 快捷工具栏
编辑器顶部提供常用Markdown语法的快捷按钮：
- **B**：插入粗体文本
- **I**：插入斜体文本
- **#**：插入标题
- **•**：插入列表项
- **@**：插入链接模板
- **{}**：插入代码标记

### 预览功能
- 点击"预览"按钮可查看Markdown渲染效果
- 再次点击"编辑"按钮返回编辑模式
- 预览模式下内容为只读状态

### 管理备忘录
- **选择备忘录**：点击左侧列表中的备忘录标题
- **删除备忘录**：点击备忘录旁的垃圾桶图标
- **时间排序**：备忘录按最后修改时间排序
- **自动保存**：编辑内容会自动保存，无需手动操作

## 数据管理

### 数据导出
1. 点击导航栏"导出数据"按钮
2. 系统会生成包含所有数据的JSON文件
3. 文件名格式：tripmaster-export-日期.json
4. 导出内容包括：兴趣点、行程、预算、备忘录

### 数据导入
1. 点击导航栏"导入数据"按钮
2. 选择之前导出的JSON文件
3. 系统会自动导入所有数据
4. 导入完成后页面会自动刷新

### 数据备份建议
- 建议每次重要规划完成后都导出数据
- 将导出文件保存到云端或多个设备
- 定期备份以防数据丢失
- 导出文件可用于跨设备同步

## 常见问题

### Q: 备忘录支持哪些Markdown语法？
A: 支持常用的Markdown语法，包括标题、粗体、斜体、列表、链接、代码等基本格式。

### Q: 备忘录内容会自动保存吗？
A: 是的，备忘录内容会在您停止输入后自动保存，无需手动操作。

### Q: 可以导出备忘录为其他格式吗？
A: 目前只支持JSON格式的整体数据导出，如需其他格式可复制内容到相应编辑器中转换。

### Q: 备忘录有字数限制吗？
A: 没有严格的字数限制，但建议单个备忘录保持合理长度以便管理。

### Q: 可以搜索备忘录内容吗？
A: 目前版本暂不支持内容搜索，建议使用有意义的标题便于查找。
