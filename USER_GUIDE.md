# 旅行规划大师使用指南

## 快速入门

### 1. 环境准备
确保您已安装：
- Node.js (版本 >= 16.0.0)
- npm (通常随Node.js一起安装)

### 2. 项目启动
```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev
```

启动成功后，访问：
- 前端界面：http://localhost:8080
- 后端API：http://localhost:3000

### 3. 配置高德地图API
编辑 `.env` 文件，填入您的高德地图API密钥：
```env
AMAP_API_KEY=your_actual_api_key_here
```

## 功能使用说明

### 🗺️ 景点筛选功能

**搜索景点：**
1. 在左上角搜索框输入景点名称（如"故宫"、"西湖"等）
2. 点击"搜索"按钮或按回车键
3. 左侧显示搜索结果列表，地图上标记所有结果点

**管理兴趣点：**
- 点击搜索结果旁的"+"按钮添加到兴趣点
- 点击右侧兴趣点列表中的"-"按钮移除兴趣点
- 点击搜索结果可在地图上定位该景点

### 📋 行程规划功能

**创建行程：**
1. 点击左侧"新建行程"按钮
2. 输入行程名称和日期
3. 从下方"可添加的景点"列表中选择景点加入行程

**编辑行程：**
- 在景点列表中可调整顺序（使用↑↓按钮）
- 为每个景点添加详细描述和预算
- 点击"保存"按钮保存行程信息

**查看路线：**
- 地图区域自动显示当前行程的路线
- 切换"编辑"/"预览"模式查看不同视图
- 预览模式以时间轴形式展示行程

### 💰 预算管理功能

**同步预算：**
1. 点击"从行程同步"按钮
2. 系统自动导入行程中的景点预算
3. 可手动添加交通、住宿等其他预算

**预算分类：**
- **景点预算**：来自行程规划的景点费用
- **交通预算**：各类交通工具费用
- **其他预算**：住宿、餐饮等自定义预算

**数据可视化：**
- 饼图显示各类预算占比
- 柱状图对比各项预算金额

### 📤 数据管理

**导出数据：**
- 点击导航栏"导出数据"按钮
- 系统生成包含所有信息的JSON文件
- 文件名为：tripmaster-export-日期.json

**导入数据：**
- 点击"导入数据"按钮选择JSON文件
- 系统自动恢复所有规划信息
- 导入后页面自动刷新

## 最佳实践建议

### 规划流程推荐
1. **收集兴趣点**：先使用景点筛选功能收集感兴趣的地点
2. **制定行程**：基于兴趣点创建具体的旅行计划
3. **预算管理**：同步并完善各项费用预算
4. **定期备份**：及时导出数据以防丢失

### 使用技巧
- 搜索时尽量使用具体地名，如"北京故宫"而非仅仅"故宫"
- 行程规划时建议按地理位置相近程度排序景点
- 预算管理要及时更新实际花费与预算的差异
- 定期导出数据作为备份

## 常见问题解答

**Q: 搜索不到想要的景点怎么办？**
A: 尝试使用更具体的关键词，或者检查高德地图API配置是否正确。

**Q: 地图不显示或标记点不准确？**
A: 确认网络连接正常，并检查API密钥是否有效。

**Q: 数据丢失了怎么办？**
A: 如果之前导出过数据，可以通过导入功能恢复。

**Q: 如何在不同设备间同步数据？**
A: 导出数据文件，然后在新设备上导入即可。
